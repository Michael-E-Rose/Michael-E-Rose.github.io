Software

pybliometrics (formerly scopus)

John Kitchin and I develop pybliometrics, being a Python module for easy, large-scale interaction with Elsevier's Scopus bibliometric database.

Software Documentation     Software repository

sosia

Stefano Baruffaldi and I develop sosia, a Python module to match researchers online in the Scopus database (useful for Diff-in-Diff analysis).

Learning from the Masters

Being an empirical researcher, I'd like to recommend a number of "manuals" that helped me set foot in the academic world:

  • "Code and Data for the Social Sciences: A Practitioner's Guide", by J. Shapiro and M. Gentzkow
  • "Writing Tips for Ph. D. Students", by J. Cochrane
  • "Some Research and Writing Tips", Part 1 (Research) and Part 2 (Writing), by B. Jacobsen
  • "Think Python: How to Think Like a Computer Scientist", by A. Downey
